# Hash Maps

Key-value store.

**Why:** Fast lookups.
**When:** Efficient mapping needed.
**How:**

JavaScript:
```js
let map = new Map();
map.set('a', 1);
map.get('a');
```

Java:
```java
Map<String, Integer> map = new HashMap<>();
map.put("a", 1);
map.get("a");
```

**Built-in Methods:**
- JS: Map(), Object
- Java: HashMap
